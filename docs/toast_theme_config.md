ToastUtils样式配置
----
#### 1.属性介绍

```json
//背影
"backgroundResource": {
    "name": "资源名称",
    "type": "drawable或mipmap"
  }

//文本颜色
"textColor": "#ffffff"

//文本大小(一般指dp)
"textSize": 12
```

#### 2.配置示例
```json
{
  "backgroundResource": {
    "name": "bg",
    "type": "drawable"
  },
  "textColor": "#ffffff",
  "textSize": 12
}
```
